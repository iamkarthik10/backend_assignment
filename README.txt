This application will populate dummy data in database and by using an api will fetch the data in JSON format

1. Name of the application is userlogin.
2. For adding the dummy data into the database use the command python manage.py Populate_User_Data name location 'datetime' 'datetime' we need to pass all in str format.
3. Then we can use the api ex: http://127.0.0.1:3001  -- to get the required JSON
